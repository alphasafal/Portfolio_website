`hero-robot.splinecode` is extracted from that public embed (self-hosted = no Spline watermark overlay).

## Re-export after editing the scene

1. Open the scene in [Spline](https://app.spline.design)
2. **Export → Code → Download** `.splinecode`
3. Replace `hero-robot.splinecode`
4. Redeploy

Or set `NEXT_PUBLIC_SPLINE_HERO_SCENE` to a `prod.spline.design/.../scene.splinecode` URL.

Source editor file: `source/robot_follow_cursor_for_landing_page_mc-2.spline`
